# Icon Attribution

The icons in this directory are based on the **Hot Tea Icon** from [UXWing](https://uxwing.com/hot-tea-icon/).

## License

[UXWing icons](https://uxwing.com/license/) are free for personal and commercial use without attribution required.

## Modifications

The original UXWing icon has been adapted for this project:
- Resized to 48x48 and 96x96 PNG formats using [MyImageTools](https://myimagetools.com) online resizing and conversion tool
- SVG version maintained with original UXWing design

## Tools Used

- [MyImageTools](https://myimagetools.com) - Online image resizing and conversion tool used to create PNG versions
